Noteworthy
This project utilizes custom elements for the header, footer, and card (on app page), and the card uses slots to insert the buttons into it.
Uses fetch to call a custom proxy server that serves both GMP API endpoints used for this project.
The Bulma framework is used in all parts of the site, even here!
I am really proud of my progress so far. All project requirements have been met and I wrote a PHP server script that can serve multiple API endpoints.
Grading
Like I said, I have met all requirements (custom elements, images, fonts, and conventions, along with Bulma, localStorage, and fetch) AND I learned some beginner PHP on the way.
I feel the input requirements have also been met, with one of the inputs being the user's geolocation, and then the type selector and the next/previous buttons being the other ones that are obvious.
